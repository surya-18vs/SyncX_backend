## API Routes

### Add Patient

POST https://syncx-backend-a5pj.onrender.com/api/patient/add

### Get All Patients

GET https://syncx-backend-a5pj.onrender.com/api/patient/all

### Call Next Patient

POST https://syncx-backend-a5pj.onrender.com/api/patient/next

### Clear Completed Patients

DELETE https://syncx-backend-a5pj.onrender.com/api/patient/clear-completed
